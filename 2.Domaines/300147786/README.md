# Application de Recherche de Colocataires

## Description
Cette application permet aux utilisateurs de **trouver des colocataires compatibles** selon leurs critères personnels.  
Elle offre la possibilité de créer un profil, publier des annonces de colocation, définir des préférences, matcher avec d’autres utilisateurs et communiquer via un système de messagerie intégré.
